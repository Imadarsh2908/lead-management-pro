# Lead Management System - Pro

A sleek, client-side, fully functional lead management system built entirely with modern HTML, CSS (Glassmorphism Dark Theme), and Vanilla JavaScript. It allows sales teams to manage their deals, visualize pipelines, and track performance metrics seamlessly.

## 🌟 Features

- **Rich Aesthetic Design**: Features a premium dark mode with glassmorphism UI elements, smooth gradients, and elegant micro-animations.
- **Kanban Pipeline**: Drag and drop leads across different sales stages (New, Contacted, Qualified, Proposal, Won, Lost) to instantly update their status.
- **Advanced Filtering & Sorting**: Quickly search and filter leads by status, priority, source, and assignee.
- **Analytics Dashboard**: Interactive bar charts and KPI metrics to track conversion rates, pipeline value, and top leads.
- **Activity & Task Tracking**: Keep a comprehensive log of activities and manage pending/overdue tasks.
- **Bulk Actions**: Update statuses or delete multiple leads simultaneously.
- **Data Persistence**: Uses browser LocalStorage to ensure your data stays intact between sessions without a backend.
- **Import & Export**: Backup and restore your complete workspace using JSON, or export leads directly to a CSV file.
- **Fully Responsive**: Works seamlessly across desktops, tablets, and mobile devices.

## 🚀 Getting Started

Since this is a purely client-side application, there is no complicated setup, package installation, or backend server required.

1. Clone or download this repository.
2. Double-click `index.html` to open it in any modern web browser (Chrome, Edge, Firefox, Safari).
3. Start managing your leads! The system comes pre-populated with sample Indian persona data and INR (₹) currencies so you can test out the features immediately.

## 📂 Project Structure

- `index.html` - The core HTML structure, modals, and UI layouts.
- `style.css` - The stylesheet containing the luxurious dark theme, custom properties, typography (Outfit font), and responsive rules.
- `script.js` - The vanilla JavaScript application logic, handling state management, kanban drag-and-drop interactions, form submissions, and local storage data persistence.

## 🛠 Tech Stack

- **HTML5**
- **CSS3** (Flexbox, CSS Grid, CSS Variables, Glassmorphism Effects)
- **Vanilla JavaScript** (ES6+)
